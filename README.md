# Alert Triage Playbooks (SOC Gold ⭐⭐⭐⭐⭐)

This repository contains **day-to-day SOC alert triage playbooks** designed for real-world Security Operations Centers, with a strong focus on **banking and enterprise environments**.

## What’s Included
- Step-by-step triage flows
- Evidence collection checklists
- Escalation criteria
- Containment & response actions
- High-signal example playbooks

## Playbooks
- Phishing Alert
- Malware Detection
- Suspicious Login
- DLP Alert (Banking-focused)

These playbooks are written in **Markdown** and are GitHub-ready.
