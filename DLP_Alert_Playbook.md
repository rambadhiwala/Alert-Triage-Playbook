## 1. Alert Overview
Data Loss Prevention alert involving sensitive data movement.

## 2. Triage Flow
1. Identify data type (PII, PCI, PHI)
2. Validate user & business justification
3. Review transfer method
4. Assess intent (negligent vs malicious)

## 3. Evidence Checklist
- File name and content classification
- Destination (email, cloud, USB)
- User role and access level
- Prior DLP alerts

## 4. Escalation Criteria
- Customer PII or PCI exposed
- External data transfer
- Insider threat indicators

## 5. Containment Actions
- Block transfer
- Suspend user access (if required)
- Notify Legal & Compliance
- Preserve forensic evidence

## 6. Notes
Critical playbook for **banking and financial institutions**.
